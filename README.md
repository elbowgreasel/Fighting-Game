# Fighting-Game

Currently allows inputs from two players.

#### - Player:

      W - Jump

      A - Move Left

      D - Move Right

      SpaceBar - Attack

#### - Enemy:

      ArrowUp - Jump

      ArrowLeft - Left

      ArrowRight - Right

      ArrowDown - Attack
